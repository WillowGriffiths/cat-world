Welcome to cat world! Not everything works.

## Dependencies

- zig 0.15.1
- GLFW 3
- OpenGL
- assimp
- ncurses
- FreeType 2
- bun

## Running

Only Linux (including WSL) is supported!

```
zig fetch

bun start
```
